SSM框架搭建

使用IDEA创建maven的web项目

###对pom.xm添加依赖

  1. 添加配置jdk的版本，maven的依赖包maven-compiler-plugin
  2. 添加spring的context、webmvc、aspects、jdbc、test包
  3. 加载mybatis包，mybatis的逆向生成包mybatis-generator-core，spring整合mybatis的mybatis-spring整合包。
  4. 加载数据库的区动包mysql-connector-java，配置数据连接池cp30、dbcp、druid中选一个
  5. 配置servlet的依赖包，web2.0使用servlet-api，web3.0使用javax.servlet-api

###往web.xml中添加spring容器，前端控制器，字符编码控制器等

  1. 使用加载spring的配置文件，
  2. 使用 配置spring的监听,在项目一启动就创建spring的容器，这个监听器实现了ServletContextListener接口
  3. 使用配置springmvc的前端控制器DispatcherServlet，加载springmvc的配置文件
  4. 使用配置请求的过滤器

添加Spring配置文件

  1. 开启spring的注解扫描
  2. 加载数据配置文件jdbc.properties，
  3. 配置数据库源dataSource，使用dbcp、cp30.druid都行
  4. 配置MyBatis的Session工厂，将session工厂交给spring管理，也就是集成mybatis
  5. 配置扫描mybatis的dao接口 ，会为dao接口创建myabtis的dao接口实现类对象，放置到session工厂中
  6. 声明spring 的事物管理器对象，声明以注解的方式配置spring 的事物

创建数据库属性文件

###创建SpringMVC配置文件

  1. 启用注解mvc:annotation-driven
  2. 配置扫描注解context:component-scan
  3. 配置视图解析器
  4. 配置MultipartResolver，用于上传文件，使用spring的CommonsMultipartResolver

创建mybatis配置文件

  1. 配置启用下划线与驼峰式命名规则的映射
  2. 配置自动扫描包路径给类配置别名
